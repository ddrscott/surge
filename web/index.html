How It Works

  • Bugs crawl across your terminal trailing red matrix wires
  • Type a bug's name to squash it before it breaches the wall
  • Kill fast and close to the wall for higher scores
  • Chain kills to build combos — every 3 kills boosts your multiplier
  • Catch power-up words for shields, slow-mo, and double points
  • Survive escalating waves — bugs get faster and more numerous

Leaderboards

Sign in with your email to submit scores to the global leaderboard. No password required.

  1. Click sign in or use the in-game link
  2. Enter your email — you'll get a magic link + 6-digit code
  3. Click the link or enter the code
  4. You're in. Scores auto-submit on game over.

Also Runs in Your Terminal

Surge is a Node.js app with zero dependencies. Clone and run it locally:

git clone https://github.com/ellipsis-tech/surge
cd surge && npm install && npm run dev
